imaginglss.analysis.tycho_veto module¶

veto objects based on Tycho catalogue. This is based on the email discussion at:

Date: June 18, 2015 at 3:44:09 PM PDT To: decam-data@desi.lbl.gov Subject: decam-data Digest, Vol 12, Issue 29

Current usage is to first create a Tycho catalogue object, then use DECAM_XXX functions to create a mask :

>>> tycho = Tycho("pathto_tycho.fit")
>>>

The return values are True for ‘preserve’, False for ‘reject’
