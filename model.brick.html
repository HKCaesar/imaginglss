model.brick module¶

class model.brick.Brick(id, name, ra, dec, ra1, ra2, dec1, dec2)[source]¶

Bases: object

The base (immutable) object corresponding to an imaging brick. This object contains only the meta-data, and it calls imagerepo to handle looking up the pixel-level information. Major methods are “readout”, “query” and “revert”.

query(repo, coord)[source]¶

Returns the xy index of pixels at coord coord can be:

(RA, DEC) tuple of arrays
array of shape (2xN) RA DEC

In either case returns a (2xN) array Note that the returned array is in a x-first-varying order. To index the ndarray image, use [y, x].

readout(coord, repo, default=nan)[source]¶

Return image values at coord=(RA, DEC) using the imagerepository repo. If the image in this brick does not cover this region, put in `default’.

revert(repo, xy)[source]¶

Returns the RA, DEC index of pixels for coord coord can be:

(x, y) tuple of arrays array of shape (2xN) x, y

In either case returns a (2xN) array.
