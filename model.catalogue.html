model.catalogue module¶

class model.catalogue.Catalogue(cachedir, filenames, aliases)[source]¶

Bases: model.utils.columnstore.DiskColumnStore

Class for handling object catalogs associated with a data release. The catalogs are contained in FITS files, but this class caches the information for speed and only columns that are accessed are loaded into memory. The columns are cached into cachedir, which is initialized by the DataRelease object. We assume the catalogue is split into multiple files. and the list of file names are provided as an input. The main method is fetch.
