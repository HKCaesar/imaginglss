class imaginglss.DECALS(filename=None)[source]¶

Bases: object

DECALS configuration object

Load DECALS configuration from a configuration file. The configuration file describes where and how to access the data release.

The main objects for access this data release are created as the configuration object is parsed.

If filename is None, the file pointed by environment variable DECALS_PY_CONFIG is used.

Attributes

datarelease (imaginglss.model.datarelease.DataRelease) The datarelease object described by this configuration file
sfdmap (imaginglss.model.sfdmap.SFDMap) The dust extinction object described by this configuration file.
