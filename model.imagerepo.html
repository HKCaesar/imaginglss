model.imagerepo module¶

class model.imagerepo.ImageRepo(root, pattern)[source]¶

Bases: object

Image repository. This class serves as an interface to the file system, the layout of files and IO for brick files. Given a Brick object, ImageRepo can return the image data or meta data of the image. Standard users should not need to modify this class.
