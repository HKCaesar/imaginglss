imaginglss.utils package¶

In order to reduce the number of dependecies, we implemented a few coordinate transformations

WCS¶

  • imaginglss.utils.wcs_tangent impments the WCS tangent format.
  • imaginglss.utils.wcs_simplezea implement the WCS ZEA format; oriented at north / south pole
  • imaginglss.utils.euler implements the transformation between Galactic and RA/DEC. coordinates.

Sharedmem¶

  • imaginglss.utils.sharedmem implements a MapReduce pool object for in-node parallellism. See sharedmem
